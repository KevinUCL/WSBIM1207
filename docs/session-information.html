10.2 R package setup

To install all necessary packages to run all the code, please execute the following code:

pkgs <- c(base, datasets, dplyr, forcats, ggplot2, graphics,
          grDevices, gridExtra, lubridate, methods, purrr, readr,
          rWSBIM1207, stats, stringr, tibble, tidyr, tidyverse, utils)
if (!require("BiocManager"))
   install.packages("BiocManager")
BiocManager::install(pkgs)
